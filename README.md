Recru filter foi um projeto desenvolvido durante um hackathon. A ideia consiste em empresas que necessitam de desenvolvedores possam cadastrar uma vaga de emprego de maneira rápida, fácil e objetiva. Dessa forma o desenvolvedor saberá exatamente com o que está lidando e se quer a vaga de fato. Além disso, o desenvolvedor poderá deixar seu interesse de vaga, para que as empresas possam encontrar.

 O mais interessante desse projeto, foi ter desenvolvido tudo isso em apenas dois dias. O que me fez perder uma noite de sono, um código não tão limpo, mas funcional. O meu maior orgulho foi ter vencido em primeiro lugar esse Hackathon.
